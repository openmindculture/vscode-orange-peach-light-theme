# vscode-orange-rain-light-theme

A colorful orange light theme for Visual Studio Code inspired by IntelliJ Orange Rain Light.

## Installation

1. Install this extension from VS Code if packaged.
2. Open the Command Palette and select `Preferences: Color Theme`.
3. Choose `Orange Rain Light`.

## Theme Files

- `package.json` publishes the theme contribution.
- `themes/orange-rain-light-color-theme.json` contains the VS Code color and token mappings.
